# Caleb Andrew Omojowo — Portfolio

Personal portfolio site for Caleb Andrew Omojowo — Cybersecurity Analyst & Web Developer.
Built with Next.js (App Router) and TypeScript.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy

Push to GitHub and import the repo on [Vercel](https://vercel.com/new).
